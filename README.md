# pgp-disc

## Run
```rust
cargo run -p app
```

## Discord
  - https://discord.com/developers/applications
  - <b>New Application</b>

### Bot Configuration
  - <b>Public Bot</b>
  - enable <b>Message Content Intent</b>

### OAuth2 Configuration (URL Generator)
  - scopes: <b>bot</b>
  - permissions:
    - <b>View Channels</b>
    - <b>Send Messages</b>
    - <b>Read Message History</b>

## Env
  - bot token
  - channel id
